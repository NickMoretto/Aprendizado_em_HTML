Conhecimentos gerais sobre HTML

DOCTYPE html = versão do site.
Protocólo = São Regras.   Programar = ensinar o computador.   Algoritimo = é o passo a passo ensinado - Tem maneira certa de escrever.
HTTP = é oque conecta você com o'que você quer.   URL = Endereço universal.   CSS = organização e cores.
FRONT-END
Oque vai ser visto no site, como uma loja.
BACK-END
É o operacional da loja, por dentro da loja.
HTML
É uma Linguagem de formatação de texto, por exemplo, o botão estou com sorte google.
Anatomia - Sempre que abre com "<" se fecha com "/>"
META
É a configuração da página.
META CHARSET
Conjunto de características. charset = utf-8 = para não desconfigurar páginaem alguns navegadores (serve para padronizar).
NAME= "viewpoint" content"width= device- width, initial-scale 1.0"
A tela vai ser do tamanho da tela do dispositivo - para jogos que tem a parte preta na tela, por exemplo, a configuração é outra(diferente de 1.0). SCALE=1.0 = escala não é maior que 1.0, ou seja, não fica maior que a tela.
Head
configuração interna do site.
body
Oque vai ser visivel no site.
